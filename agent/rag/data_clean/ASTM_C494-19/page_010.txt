Therefore, results of two properly conducted tests by the same operator on the same material are not expected to differ by more than 0.275 %.

X1. SUMMARY OF TESTING PROGRAM
X1.1 The testing program to demonstrate compliance with this specification requires testing specimens made from the test
concrete that includes the admixture and comparing the aver-
age of the test results with those obtained from specimens
made from the reference concrete. For compliance, all criteria
and Table 1 have to be satisfied. The testing program requires
making at least three batches of the reference concrete and at least three batches of each test concrete, if more than one
admixture is being evaluated in the same testing program. X1.2 Fig. X1.1 is a schematic of the minimum testing

program involving one test concrete. On any given day, the
same number of batches of test concrete and reference concrete should be made. For example, if one test concrete is involved,
on a given day it is not permitted to make one batch of the reference concrete and two batches of the test concrete. The results for the individual specimens of each concrete category are averaged as shown in Fig. X1.1. The average for the test
concrete specimens (7) is compared with the average for the
reference concrete specimens (R).
